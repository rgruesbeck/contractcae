# contractcae.com
http://contractcae.com.s3-website-us-west-2.amazonaws.com/

Website for contractcae.com
```sh
npm install
```

## Get Started
Start gulp build
```sh
npm start
```
Open a new shell and start the server
```sh
npm run server
```

## Deploy
```sh
npm run deploy
```
